This page shows how to use the API behind: http://tsviewer.vito-eodata.be
Use your browsers developer tools to inspect the code, and the queries being sent.

General info about the Saiku REST API is available at: http//tsviewer.vito-eodata.be/saiku4/serverdocs/
You can also use the online Saiku webapp to explore the available cubes, and to easily create MDX queries (login with username 'admin' and password 'admin').

The first query simply logs in with saiku, and creates a session. Once we have a session, we have to create a new query. Saiku returns this result when the query is created:
Before we can start constructing MDX queries, we might want to query some metadata about the OLAP cube. For instance, we can query the list of available countries.
This simple GET request returns the list below: 
http://localhost:9000/saiku4/rest/saiku/admin/discover/TimeSeriesWithRegionsHadoop/ESECube/ESECube/EseCube/dimensions/Geography/hierarchies/geography/levels/Country
Finally, we can use the constructed query to send as many MDX queries as we like. MDX is a query language, like SQL, but specifically designed for OLAP cubes.
